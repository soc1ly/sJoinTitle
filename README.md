sJoinTitle — легкий плагин для отображения приветственных титров игрокам на вашем сервере Minecraft. Автоматически показывает настраиваемый заголовок и подзаголовок игроку, входящему на сервер<br>

Требования:<br>
› Java 16 или новее<br>
› Версия сервера 1.16 - 1.21+<br>

Конфигурация (config.yml): <br>
title — основной заголовок титра (поддерживает цветовые коды &)<br>
subtitle — подзаголовок титра (поддерживает цветовые коды &)<br>

Пример:<br>
[CODE=yaml]title: '&a[COLOR=rgb(97, 189, 109)]Приветствуем![/COLOR]'<br>
subtitle: '&7[COLOR=rgb(124, 112, 107)]Это сообщение, которое появляется при входе на сервер[/COLOR]'[/CODE]<br>

Особенности:<br>
› Поддержка цветовых кодов<br>
› Минимальная нагрузка на сервер<br>
