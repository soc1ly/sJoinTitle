sJoinTitle — легкий плагин для отображения приветственных титров игрокам на вашем сервере Minecraft. Автоматически показывает настраиваемый заголовок и подзаголовок игроку, входящему на сервер

Требования:
› Java 16 или новее
› Версия сервера 1.16 - 1.21+

Конфигурация (config.yml): 
title — основной заголовок титра (поддерживает цветовые коды &)
subtitle — подзаголовок титра (поддерживает цветовые коды &)
Пример:
[CODE=yaml]title: '&a[COLOR=rgb(97, 189, 109)]Приветствуем![/COLOR]'
subtitle: '&7[COLOR=rgb(124, 112, 107)]Это сообщение, которое появляется при входе на сервер[/COLOR]'[/CODE]

Особенности:
› Поддержка цветовых кодов
› Минимальная нагрузка на сервер
